## 🛠️ Quick Setup (အမြန်သွင်းနည်း)

အောက်ပါ Command ကို Copy ကူးပြီး သင်၏ VPS Terminal တွင် Run ပါ။

```bash
# System Update လုပ်ခြင်း
sudo apt update && sudo apt upgrade -y

# Marzban Script အသုံးပြု၍ Install လုပ်ခြင်း
sudo bash -c "$(curl -sL [https://github.com/Gozargah/Marzban-scripts/raw/master/marzban.sh](https://github.com/Gozargah/Marzban-scripts/raw/master/marzban.sh))" @ install
```
